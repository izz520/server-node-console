# 任务记录：节点页服务器状态自动刷新

## 任务状态

已完成

## 任务背景

节点页安装表单现在只允许选择 `normal` 状态服务器。服务器状态可能在服务器页面通过 SSH 测试后变化，如果节点页的服务器列表不刷新，用户可能看不到刚恢复正常的服务器。

## 本次任务目标

为节点页服务器列表增加轻量轮询，让安装表单中的可选服务器状态保持更新。

## 不包含范围
