# 任务记录：服务器管理与 SSH 连通性测试

## 任务状态

已完成

## 任务背景

根据 `PRD.md`，用户完成注册和登录后，下一步需要添加自己的服务器。服务器是后续安装 sing-box 协议节点、维护 NAT 端口映射和生成订阅的基础资源。因此本任务优先实现服务器管理能力，并完成 SSH 凭据加密保存和连通性测试。

## 本次任务目标

实现普通用户对自己服务器的增删改查、SSH 密码/私钥加密保存、凭据脱敏展示，以及手动/保存前 SSH 连通性测试。

## 本次调整范围

### 后端

- 实现服务器列表接口：`GET /api/v1/servers`
- 实现服务器详情接口：`GET /api/v1/servers/:id`
- 实现添加服务器接口：`POST /api/v1/servers`
- 实现编辑服务器接口：`PUT /api/v1/servers/:id`
- 实现删除服务器接口：`DELETE /api/v1/servers/:id`
- 实现 SSH 连通性测试接口：`POST /api/v1/servers/:id/test-ssh`
- 添加服务器时执行保存前 SSH 连通性测试。
- SSH 密码和 SSH 私钥使用 `ENCRYPTION_KEY` 加密后保存。
- 返回服务器信息时不返回 SSH 密码或私钥明文，只返回凭据是否已保存。
- 所有服务器接口按当前登录用户隔离数据。
- 管理员仍保持只读能力，本任务不实现管理员服务器列表。

### 前端

- 实现服务器管理页面。
- 展示当前用户自己的服务器列表。
- 支持添加服务器。
- 支持编辑服务器基础信息和重新填写 SSH 凭据。
- 支持删除服务器。
- 支持手动点击测试 SSH 连通性。
- 表单支持密码认证和私钥认证两种方式。
- 页面展示服务器状态：正常、连接失败、禁用。
- 页面不展示 SSH 密码或私钥明文。

## 不包含范围

- 不实现 NAT 端口映射管理。
- 不实现协议节点安装。
- 不实现任务队列系统。
- 不实现管理员服务器只读列表。
- 不实现服务器删除时的复杂关联校验；当前没有节点/订阅业务，可先支持删除。
- 不实现 SSH 凭据明文查看。

## 验收标准

- 用户可以添加密码认证服务器。
- 用户可以添加私钥认证服务器。
- 添加服务器时后端会进行 SSH 连通性测试。
- SSH 测试成功后服务器状态为 `normal`。
- SSH 测试失败时接口返回失败原因，服务器不保存或保存为连接失败状态，需本任务确认。
- 用户可以查看自己的服务器列表。
- 用户不能查看或操作其他用户的服务器。
- 用户可以编辑自己的服务器基础信息。
- 用户可以重新填写 SSH 凭据并覆盖旧凭据。
- 用户可以手动触发 SSH 连通性测试。
- 后端响应不包含 SSH 密码或私钥明文。
- 前端服务器页面可以完成列表、添加、编辑、删除、测试连通性操作。
- `go test ./...`、`pnpm check`、`pnpm build` 通过。

## 风险与待确认

- 添加服务器时，如果 SSH 测试失败，建议不保存服务器，并返回失败原因。
- SSH 私钥格式和 passphrase 支持范围需要确认；建议第一期先支持无 passphrase 私钥。
- `ENCRYPTION_KEY` 当前示例值长度不一定适合真实生产加密，第一期可先校验长度并给出明确错误。
