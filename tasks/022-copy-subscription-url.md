# 任务记录：订阅链接一键复制

## 任务状态

已完成

## 任务背景

`PRD.md` 的创建订阅流程要求用户创建订阅后复制订阅链接到客户端使用。当前订阅列表只展示 `/sub/{token}` 链接文本，没有一键复制操作，用户体验不完整。

## 本次任务目标

在订阅列表中为每个订阅链接增加复制按钮，并在复制成功/失败后给出页面反馈。

## 本次调整范围

### 前端

- 订阅卡片中增加复制链接按钮。
- 使用浏览器 Clipboard API 写入订阅链接。
- 复用页面 message 展示复制结果。

## 不包含范围

- 不生成二维码。
- 不调整订阅 token 规则。
- 不改后端接口。

## 验收标准

- 用户可以点击按钮复制订阅链接。
- 复制成功后页面显示反馈。
- `pnpm check`、`pnpm build` 通过。

## 风险与待确认

- Clipboard API 在 localhost 和 HTTPS 下可用；非安全上下文可能被浏览器拒绝。

## 完成记录

- 订阅卡片已增加复制订阅链接按钮。
- 复制时会补齐当前站点 origin，生成完整订阅地址。
- 复制成功/失败会通过页面 message 给出反馈。

## 验证结果

- `pnpm check` 通过。
- `pnpm build` 通过。
