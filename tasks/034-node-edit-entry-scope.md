# 任务记录：节点编辑入口范围收紧

## 任务状态

已完成

## 任务背景

后端当前只允许编辑外部导入节点，系统安装节点的参数来自安装任务和脚本执行结果，暂不支持直接编辑。但前端节点列表对所有节点都展示编辑按钮，用户点击系统安装节点会遇到后端错误。

## 本次任务目标

前端节点列表只对外部导入节点展示编辑按钮，避免用户触发当前版本不支持的操作。

## 本次调整范围

### 前端

- 系统安装节点隐藏编辑按钮。
- 外部导入节点保留编辑能力。

## 不包含范围

- 不实现系统安装节点参数编辑。
- 不改变后端接口。

## 验收标准

- 外部节点可编辑。
- 系统安装节点不显示编辑按钮。
- `pnpm check`、`pnpm build` 通过。

## 风险与待确认

- 后续如果支持安装节点编辑/重试，可以重新开放入口。

## 完成记录

- 系统安装节点不再展示当前版本不支持的编辑入口。
