# 任务记录：安装表单默认参数文案同步

## 任务状态

已完成

## 任务背景

后端已经支持安装协议时自动生成端口和 UUID，但前端安装表单仍显示“留空由脚本随机”“UUID 可选”等较模糊文案。需要让用户清楚知道这些参数留空时由系统自动生成。

## 本次任务目标

更新协议安装表单占位文案，和当前后端默认参数生成逻辑保持一致。

## 本次调整范围

### 前端

- 端口输入框文案改为留空后端自动生成。
- UUID 输入框文案改为留空后端自动生成。
- 不改变字段结构和提交逻辑。

## 不包含范围

- 不在前端生成 UUID。
- 不展示生成后的敏感参数明文。

## 验收标准

- 安装表单文案准确描述默认参数行为。
- `pnpm check`、`pnpm build` 通过。

## 风险与待确认

- 后续如果按协议展示不同参数模板，需要再细化字段说明。

## 完成记录

- 端口输入框占位文案已改为“留空由后端自动生成”。
- UUID 输入框占位文案已改为“留空由后端自动生成”。

## 验证结果

- `pnpm check` 通过。
- `pnpm build` 通过。
