# 任务记录：移动端导航补齐

## 任务状态

已完成

## 任务背景

`AGENT.md` 要求前端考虑响应式设计，适配不同设备。当前控制台侧边栏在 `lg` 以下隐藏，但没有替代的移动端导航，用户在手机或窄屏下进入系统后无法方便切换服务器、节点、订阅、任务、安全等模块。

## 本次任务目标

补齐移动端导航，让普通用户和管理员在窄屏下也能访问核心功能页面。

## 本次调整范围

### 前端

- 在 `AppLayout` 增加移动端底部导航。
- 复用现有导航配置和管理员权限判断。
- 仅展示图标和短标签，避免窄屏文字溢出。
- 为主内容增加底部留白，避免被底部导航遮挡。

## 不包含范围

- 不重做桌面端侧边栏。
- 不新增抽屉菜单。
- 不调整各业务页面布局。

## 验收标准

- 移动端宽度下可以切换核心页面。
- 管理员移动端可看到管理员入口。
- 桌面端导航保持不变。
- `pnpm check`、`pnpm build` 通过。

## 风险与待确认

- 第一版采用底部导航；如果后续模块继续增加，可能需要改为抽屉式导航。

## 完成记录

- 已新增移动端底部导航。
- 已复用桌面端导航配置和管理员权限判断。
- 主内容已增加移动端底部留白，避免被导航遮挡。
- 桌面端侧边栏保持不变。

## 验证结果

- `pnpm check` 通过。
- `pnpm build` 通过。
