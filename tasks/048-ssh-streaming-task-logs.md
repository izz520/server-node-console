# 任务记录：安装任务增加 SSH 流式日志

## 任务状态

已完成

## 任务背景

真实 AnyTLS 安装验证中，远端 argosbx 脚本多次长时间运行，但任务日志只有开始、生成命令和最终输出。当前实现必须等 SSH 命令结束后才写入远端输出，导致用户在安装过程中无法判断是否卡住、正在下载，还是脚本进入等待。

## 本次任务目标

让 SSH 命令执行期间按行回传远端 stdout/stderr，并实时写入任务日志。安装任务长时间运行时，前端任务详情应能看到远端脚本输出进度。

## 本次调整范围

### 后端

- 扩展 `sshclient.RunCommand`，支持可选输出回调。
- 在安装/卸载任务执行 SSH 命令时，将远端输出按行写入任务日志。
- 安装任务流式日志继续复用敏感参数脱敏逻辑。
- 增加输出分行回调单元测试。

### 验证

- 运行 `go test ./...`。

## 不包含范围

- 不新增 WebSocket/SSE。
- 不改变前端轮询逻辑。
- 不修改 argosbx 脚本。

## 验收标准

- SSH 命令执行期间可以实时写入任务日志。
- 输出没有换行结尾时也能在命令结束/超时时 flush。
- `go test ./...` 通过。

## 执行结果

- 已新增 `RunCommandWithOutput`，支持远端 stdout/stderr 按行回调。
- 安装/卸载任务会将远端输出实时写入任务日志。
- 安装任务流式日志会继续脱敏 UUID 和 Argo token。
- 已补充分行、CRLF、无结尾换行 flush、空行跳过测试。
- 后端测试通过。
